CSS Box Model

Css punya height dan width untuk mengatur ukuran elemen. Namun, ukuran elemen tidak hanya terdiri dari height dan width saja, tetapi juga melibatkan padding, border, dan margin.

Box model terdiri dari:

  • Content: Area tempat konten berada, seperti teks atau gambar.
  • Padding: Ruang antara konten dan border. Padding dapat memiliki warna latar belakang yang sama dengan konten.
  • Border: Garis yang mengelilingi padding dan content. Border dapat memiliki warna, gaya, dan ketebalan yang berbeda.
  • Margin: Ruang di luar border yang memisahkan elemen dari elemen lainnya. Margin tidak memiliki warna latar belakang.
